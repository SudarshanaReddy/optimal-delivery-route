# optimal-delivery-route
--------------------------
Tech Stack Used
---------------
Java 8 <br/>
Spring Boot <br/>
Maven <br/>
Liquibase <br/>
Spring Data JPA <br/>
H2 In Memory Database
 
 Build
 -----
 mvn clean install -DskipTests
 
 Run
 ---
 mvn spring-boot:run or run it directly as SPRING BOOT APPLICATION from main class.
